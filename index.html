Awesome busy animation

Easy-to-use jQuery plugin 

 $("canvas").loadStripeJs("start");

How does it work

A stripe consists of a background called Gauss (see Normal distribution).
Combined with small rectangles it yields a great animation.

Create your own loader

gaussStop lsj-gauss-stop Stop point of the Gauss.
gaussTimeMove lsj-gauss-time-move Time before the Gauss reaches the stop point
gaussTimeVanish lsj-gauss-time-vanish Time for the Gauss to vanish
rectEnable lsj-rect-enable Show small cute rectangles
rectRandom lsj-rect-random Rectangles appear at random positions each time
rectWidth lsj-rect-width Width of rectangles
rectInterval lsj-rect-interval Time interval between consequent rectangles
rectTimeMove lsj-rect-time-move Time for rectangle to fly from left to right
rectSlowdown lsj-rect-slowdown How much rectangles are slowed down by the Gauss

Commands 

$("canvas").loadStripeJs("start");      // start animation.

$("canvas").loadStripeJs("stop");       // stop animation.

$("canvas").loadStripeJs("remove");     // remove animator instantly and completely.
                                        // the canvas gets cleared.

$("canvas").loadStripeJs({ ... });      // set options.

Examples

You can specify options either by attribute: 
<canvas id="busy"
        lsj-background-color="#F7DB7D"
        lsj-gauss-weight="0.90"
        width='800'                     // number of pixel in stripe
        height='1'                      // should always be 1
        style='width:800px;height:5px'  // this is how you'll see it
></canvas>
or through JavaScript: 
$("canvas#busy").loadStripeJs({ backgroundColor: "red" });
Also you can chain calls to loadStripeJs function: 
// override background and start
$("canvas#busy").loadStripeJs({ backgroundColor: "red" }).loadStripeJs("start")
